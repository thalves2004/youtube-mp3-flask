## 📝 Descrição

Este é um projeto em Python usando **Flask** para baixar vídeos do YouTube em formato MP3.  
O objetivo é **aprender e praticar** Flask, manipulação de arquivos e integração com bibliotecas externas como `yt_dlp`.

---

## ⚙️ Funcionalidades

- Baixar áudio de vídeos do YouTube em **MP3**  
- Armazenar histórico dos últimos downloads  
- Visualizar títulos e miniaturas dos vídeos baixados  
- Interface web simples e responsiva  

---
