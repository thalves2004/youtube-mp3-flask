# YouTube para MP3 (Flask)

Este é um projeto em **Python/Flask** que permite baixar vídeos do YouTube como arquivos **MP3** diretamente pelo navegador.

---

## Funcionalidades

- Baixar vídeos do YouTube em MP3.
- Histórico com até 5 downloads recentes.
- Exibição da capa (thumbnail) do vídeo.
- Interface simples e responsiva.

---

## Pré-requisitos

- Python 3.8 ou superior
- [FFmpeg](https://ffmpeg.org/download.html) (necessário para conversão para MP3)
- Pip (gerenciador de pacotes Python)
